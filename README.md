webapp
======

###Gulp Sass Browser-Sync

This is a simple boilerplate for a fresh front-end project


###Usage###

```
gulp watch
```

```
gulp build
```

###Setup###

####(1) First install Node.js
####(2) 

```
npm install -g gulp
```
####(3) 

```
npm install -g bower
```
####(2) switch to your directory and install dependencies

```
npm install
```

```
bower install
```
